# Система антиплагиата

Микросервисная система для проверки студенческих работ.

## Состав системы
- **ApiGateway** (порт 5000) - основной API
- **FileService** (порт 5001) - хранение файлов
- **AnalysisService** (порт 5002) - анализ на плагиат

## Запуск
```bash
docker-compose up -d
```

## Основные endpoints
- `POST /api/files/upload-with-analysis` - загрузить работу
- `GET /reports/{workId}` - получить отчеты

Также Swagger работает и для микросервисов - при желании можно обособленно проверить работу каждого из них

## Алгоритм проверки
1. Извлекаем слова из текста работ
2. Приводим к нижнему регистру, удаляем пунктуацию
3. Считаем процент совпадения уникальных слов
4. Если сходство ≥ 80% — работа считается плагиатом

Формула:
`сходство = (общие_уникальные_слова / все_уникальные_слова) × 100%`

## Swagger
- ApiGateway: http://localhost:5000/swagger
- FileService: http://localhost:5001/swagger
- AnalysisService: http://localhost:5002/swagger
